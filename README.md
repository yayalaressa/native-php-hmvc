# Native PHP HMVC
Source code untuk Native PHP HMVC

## Prerequisites/Requirements

- PHP 7.3 or greater

## Installation

Download lalu install ke localhost

## Demo

Link halaman utama
```
https://localhost/
```
Lalu link halaman admin untuk module 'welcome'
```
https://localhost/admin/welcome/dashboard
```
